# NGDT — Geekcom Deck Tools (No GUI)

Утилита для обновления SteamOS на Steam Deck в России.

## Зачем это нужно?

Роскомнадзор заблокировал серверы Valve, из-за чего `steamos-update`
перестаёт работать — дек не может получить обновления системы.

NGDT решает эту проблему: автоматически поднимает временный туннель,
запускает обновление через него, затем туннель закрывается сам.

## Установка

Подключите USB-клавиатуру к Steam Deck, перейдите в TTY
(Ctrl+Alt+F4), войдите как пользователь `deck` и выполните:

```bash
curl -fsSL https://gdt.geekcom.org/tui | bash
```

Скрипт сделает всё сам: скачает нужные файлы, запросит пароль sudo
и запустит обновление. После успешного обновления перезагрузите дек:

```bash
sudo reboot
```

## Требования

- Steam Deck с SteamOS 3.x
- USB-клавиатура (Bluetooth не работает в TTY)
- Режим TTY: Ctrl+Alt+F4
- Пароль пользователя deck (если не задан — скрипт предложит создать)

## Как это работает

```
Скрипт установки
    ↓
Запрашивает временный туннель у сервера Geekcom
    ↓
Поднимает туннель
    ↓
Запускает steamos-update через туннель
    ↓
Туннель закрывается автоматически
```
